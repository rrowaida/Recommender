Project Outline: The objective of this project is to study how to implement a real time recommendation system works. Making an efficient recommendation system involves handling a lot of data at a time. It takes a lot of computation to figure out the relevant data and make a decision out of them. Making a real time recommendation system is even harder. Parallel computing is a way of making the computation faster so that the system can produce real time output. Therefore, this project will make a real time recommendation system for movies.
